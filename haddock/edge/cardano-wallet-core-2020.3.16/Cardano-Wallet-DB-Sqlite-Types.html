Cardano.Wallet.DB.Sqlite.Types

Description

This module contains instances and types necessary for storing wallets in a SQL database with Persistent.

It's in a separate module due to the GHC stage restriction.

The ToJSON/FromJSON and Read instance orphans exist due to class constraints on Persistent functions.

artificialEpochLength :: EpochLength Source #

As a short-to-medium term solution of persisting SlotId, we use flatSlot with an artificial epochLength. I.e. /not the same epochLength as the blockchain/. This is just for the sake of storing the epoch number and slot number inside a single 64-bit field.
