Description

This module provides a Servant client for the cardano-wallet V2 API.

The functions in this module can be run with "Servant.Client.runClientM".

walletClient :: forall t. (DecodeAddress t, EncodeAddress t) => WalletClient t Source #

Produces a WalletClient for the cardano-wallet V2 API.

You should apply a Cardano.Wallet.Primitive.AddressDerivation.NetworkDiscriminant type parameter to this to get a concrete client.
