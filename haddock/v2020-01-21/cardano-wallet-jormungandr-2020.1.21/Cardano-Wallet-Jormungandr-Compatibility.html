Description

Contains various implementation decision that are specific to a particular network / protocol. This allows us to easily select a particular backend (Byron, Shelley-Rust, Shelley-Haskell) and isolate the bits that vary between those backends.

softTxMaxSize :: Quantity "byte" Word16 Source #

Jörmugandr's chain parameter doesn't include a transaction max size. The actual hard-limit for the size is constrained by the binary format and numbers used to represent the number of inputs and outputs (Word8), yet there's also a soft-limit of 8kb which results in much smaller transactions in the end.
